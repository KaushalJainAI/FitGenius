# FitGenius AI Backend

This is the Python/Django backend for the FitGenius AI recommendation platform.
It powers user profiles, secure authentication, and a robust machine-learning recommendation engine built on a consolidated dataset of 30 physical and lifestyle features.

## Project Structure

*   **`notebooks/`**: Contains the Jupyter Notebooks aligning with the Recommender Systems (RS) university course syllabus.
    *   *See [notebooks/README.md](notebooks/README.md) for details on syllabus mapping, Hybrid/Cascade models, Evaluation Metrics, and training the backend model pipeline.*
*   **`data/`**: Expected location for `merged_fitness_data.csv`.
*   **`recommendations/models/`**: Stores the compiled `preprocessor.pkl`, `knn_model.pkl`, `svd_model.pkl` generated by the backend pipeline.
*   **`fitness_backend/`**, **`users/`**, **`profiles/`**: Core Django applications.

## Machine Learning Pipeline

Before running the server for the first time, ensuring your dataset is processed:

```bash
# Activate your environment
venv\Scripts\activate

# Train and build the Recommendation Component Files
python notebooks/train_model.py
```

*Note: If `merged_fitness_data.csv` is missing from the `data/` folder, `train_model.py` will temporarily execute on automatically synthesized scaffolding data to prevent any backend view crashes until your CSV is available.*
